Descripción
Este es el frontend del proyecto EFI, desarrollado en React con Vite y Material-UI, que consume la API Flask para manejar autenticación y CRUD de usuarios/posts.

Los endpoints del backend deben estar corriendo para que la app funcione correctamente.

Notas
Requiere Node.js y npm instalados.

No subir node_modules al repositorio.

Asegurarse de que la API Flask esté levantada antes de usar el frontend.
